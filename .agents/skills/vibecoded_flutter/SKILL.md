---
name: Vibecoded Flutter Development
description: Guidelines for generating premium, aesthetically pleasing cross-platform Flutter UIs.
---

When the user requests a "vibecoded" element or when building out the Sourdough UIs, you must:
1. **Modern Aesthetics**: Utilize curated `.dark` color schemes, glassmorphism, or modern flat designs instead of default Material baseline colors.
2. **Animations**: Add subtle micro-interactions to buttons, route transitions, and state changes (use `AnimatedContainer`, `Hero`, or explicit animations).
3. **Responsiveness**: Ensure the UI scales dynamically for mobile, web, and desktop layouts.
4. **Code Quality**: Write modular, reusable widgets complying fully with `very_good_analysis`.
