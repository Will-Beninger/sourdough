# Sourdough

A vibecoded cross-platform utility app built with Flutter.

## Overview
This application will grow into a mature utility application. Currently, it features a cross-platform infrastructure supporting Web, Android, iOS, Windows, macOS, and Linux.

## Getting Started

1. **Install Dependencies:**
   ```bash
   flutter pub get
   ```
2. **Run the App:**
   ```bash
   flutter run
   ```
3. **Linting Context:**
   We enforce strict linting using `very_good_analysis`. Run `flutter analyze` to ensure code quality.

## AI Agent Integration
This repository uses `.geminirules` to manage interactions with AI coding assistants (e.g., Gemini/Antigravity). See `.geminirules` for specific VCS and formatting constraints.
